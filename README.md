# electron-vue-starter

## Project setup
```
npm install
```

### Compiles and watches files for changes
```
npm run watch
```

### Compiles and minifies for production
```
npm run build
```

### Runs Electron app
```
npm run start
```

### Lints and fixes files
```
npm run lint
```
